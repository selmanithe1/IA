Analyse du Code : KNN (\(k=5\))

Justification Technique

Le KNN base sa prédiction sur la distance euclidienne entre deux vecteurs de capteurs :

\[d(x,y) = \sqrt{\sum (x_i - y_i)^2}\]

L'astuce technique : Le choix de \(k=5\) est un compromis mathématique entre la Variance (sensibilité au bruit) et le Biais (manque de précision).

Détails de l'implémentation

  • X_scaled * 1e12 : Indispensable pour que le réseau "voit" les signaux.
  • n_jobs=-1 : Pour accélérer les calculs sur l'ordinateur de Madame.

# --- Le scaling par un billion (10^12) ---
# Sans cela, les reseaux de neurones ne voient aucune difference.

# --- Initialisation KNN ---
