Analyse du Code : Lasso (\(L_1\))

Justification Technique

L'activité du cerveau est très "sparse" (peu de zones actives). Le Lasso minimise :

\[ \min_Z \frac{1}{2} \|X - LZ\|^2_2 + \alpha \|Z\|_1 \]

La pénalité \(\|Z\|_1\) force les coefficients inactifs à valoir exactement zéro car sa forme géométrique en diamant favorise les solutions sur les axes.

Pourquoi alpha=0.5 ?

Le paramètre \(\alpha\) détermine la force de cette pénalité pour obtenir un modèle "sparse" afin de respecter la biologie cérébrale.


# --- Le Lasso pour la Sparsite ---
from sklearn.linear_model import Lasso
# alpha=0.5 determine la force de la regularisation L1
lasso = Lasso(alpha=0.5).fit(X_train, Z_train)

# Prediction seuillee pour le binaire
Z_pred = (lasso.predict(X_test) > 0.5)
