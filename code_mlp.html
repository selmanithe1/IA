Analyse du Code : MLP (Deep Learning)

L'Architecture Gagnante

Le signal traverse un entonnoir de neurones : 204 (Entrée) \(\rightarrow\) 200 \(\rightarrow\) 100 \(\rightarrow\) 450 (Sortie).

ReLU (f(x)=max(0,x)) : imite le comportement biologique. Le neurone ne déclenche un signal que si le potentiel dépasse un seuil.

Optimisation Adam & Backprop

Utilise 500 itérations de rétropropagation pour ajuster les poids et apprendre la "carte de gain inverse" du cerveau.


# --- MLP robuste avec 500 iterations ---
from sklearn.neural_network import MLPRegressor

mlp = MLPRegressor(
    hidden_layer_sizes=(200, 100), 
    activation='relu', 
    solver='adam', 
    max_iter=500,
    verbose=True
).fit(X_train, Z_train)

# Evaluation finale via F1-Score sur seuil 0.5
Z_pred = (mlp.predict(X_test) > 0.5)
