Introduction au Problème Inverse

Le projet MEG (Magnétoencéphalographie) repose sur une équation physique fondamentale qui régit la capture des signaux cérébraux :

\(X = L \cdot Z + n\)
  • \(X\) (Mesures) : Les données captées par les 204 capteurs du casque.
  • \(L\) (Matrice Lead Field) : La matrice de gain qui définit comment le champ magnétique se propage dans la tête.
  • \(Z\) (Sources) : L'activité réelle des neurones sur les 450 parcelles du cerveau (ce que nous cherchons).
  • \(n\) (Bruit) : Les interférences magnétiques ambiantes.

L'objectif de ce travail pour Mme Valérie GAUTARD est de trouver un algorithme capable d'inverser cette équation pour retrouver \(Z\) à partir de \(X\).

Analogie : "Dis-moi qui sont tes voisins, je te dirai qui tu es".
Détail technique : Compromis Biais-Variance (\(k=5\)).

Analogie : "Le sculpteur sélectif". On efface tout ce qui n'est pas essentiel.
Détail technique : Norme \(L_1\) forçant la sparsité à zéro.
